SpendTrack (Expense Tracker)

The primary objective of this project is to provide users with a centralized platform to organize their financial records, gain insights into their spending habits, and make informed budgeting decisions.The application provides a simple and intuitive interface for recording daily transactions, categorizing expenses, and monitoring overall financial health through real-time balance calculations.

A responsive expense tracker built using:
  - HTML5
  - CSS3 (Custom Design System)
  - JavaScript
  - LocalStorage API

Features
  - Add / Edit / Delete expenses
  - Category filtering
  - Monthly summary chart
  - Dark mode
  - Local storage persistence

Key Learning Outcomes

Through this project, I gained practical experience in:

Building responsive web interfaces
Developing REST-like backend functionality using Flask
Implementing CRUD (Create, Read, Update, Delete) operations
Working with relational databases using SQLite
Managing user authentication and sessions
Connecting frontend components with backend services
Performing real-time financial calculations and data processing.
